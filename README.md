# 📊 HR Data Analysis with Python

A complete **Exploratory Data Analysis (EDA)** project on an HR dataset using **Python**. This project analyzes employee information to uncover valuable workforce insights such as employee status, department distribution, salary trends, work modes, hiring patterns, performance ratings, and more.

## 🚀 Project Overview

This notebook demonstrates how Python can be used to clean, analyze, and visualize HR data to support data-driven business decisions. The analysis focuses on identifying workforce trends and generating actionable insights through descriptive statistics and visualizations.

## 📌 Key Objectives

- Analyze employee status distribution
- Explore work mode (On-site vs Remote)
- Department-wise employee distribution
- Calculate average salary by department
- Identify highest-paying job titles
- Analyze hiring trends over the years
- Study employee age distribution
- Examine performance ratings
- Explore salary distribution
- Generate meaningful HR insights using visualizations
